//Claude AI Usage

Claude was primarily used as a guidance and review agent during project development.

Usage:
- Assisted with overall project architecture and file organization.
- Suggested improvements to database schema and API design.
- Recommended best practices for frontend React component structure.
- Reviewed code readability and maintainability suggestions.

//Safety Measures:
1. Claude outputs were treated as suggestions only.
2. Manual review and testing were performed on every suggestion.
3. No secrets or sensitive data were ever shared with the AI.
4. AI was only used to accelerate development and ensure clarity, not for final decision-making.
